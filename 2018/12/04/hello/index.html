在ubuntu上搭建wordpress

1. 安装apache2

sudo apt-get install apache2 -y

进行资源更新，在终端输入：
sudo apt-get update

进行包更新，输入：
sudo apt-get upgrade -y

2. 安装PHP及相关组件

3. 安装mysql
