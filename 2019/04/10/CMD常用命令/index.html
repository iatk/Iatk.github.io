CMD常用命令

2019-04-10

前言

windows下的命令虽然不是经常用，但有些又是必须会的，每次用的时候就各种查资料，很浪费时间，不如整理一下以后用了也好翻阅。

控制台

1、进入某个盘 D: //进入D盘
2、查看目录 dir
3、创建目录和删除目录
创建目录：md 目录名（文件夹）
删除目录：rd 目录名（文件夹）
4、清除屏幕
cls 类似于linux下的clear
5、复制和移动文件（必须cd到文件目录下）
copy 文件 路径/文件名
move 文件 路径/文件名
xcopy ->复制整个文件夹的内容 用法：xcopy+文件夹+目标文件夹+参数
6、删除文件
del 文件名 （这个是专门删除文件的，不能删除文件夹）
7、管理自身的路由转发表（增删改查）
route
8、查看主机存储的IP和MAC的映射
arp
9、查看当前进程列表
tasklist
10、查看进程
query
11、查看网络状态
netstat （不间断输出，可以用CTRL+C中断）
12、网络配置
netsh （后面需要接参数）
13、查看计算机的用户
net user

WIN+R快捷键

1、远程连接
mstsc
2、配置启动项
msconfig
3、打开服务
services.msc
4、打开设备管理
devmgmt.msc
5、打开控制面板
control
6、打开注册表
regedit
7、打开删除程序的面板
appwiz.cpl
8、打开计算器
calc
9、新打开一个文本
notepad
